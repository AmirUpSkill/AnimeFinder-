# API Contract

## Overview
This document defines the API contract for the AnimeFinder semantic search application.

## Base URL
```
http://localhost:8000/api/v1
```

## Endpoints

### 1. Health Check
**GET** `/health`

### 2. Ingest Animal
**POST** `/api/v1/animals`

### 3. Get Default Animals
**GET** `/api/v1/animals`

### 4. Semantic Search
**POST** `/api/v1/search`
